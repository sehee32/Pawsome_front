#  Pawsome Front

이 프로젝트는 React를 사용하여 반려동물 용품 쇼핑몰 웹페이지를 구현했습니다.

<br><br><br>

## 🛠 기술 스택

- Frontend : React, Axios
- Backend : Spring Boot, MySQL, JPA, JWT

## 🚀 프로젝트 설정 및 실행

### 설치

```bash
npm install
```

### 실행

```bash
npm start
```

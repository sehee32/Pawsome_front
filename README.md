#  Pawsome Front

이 프로젝트는 React를 사용하여 반려동물 용품 쇼핑몰 웹페이지를 구현했습니다.

<br><br><br>

## 🛠 기술 스택

- React
- React Router
- Axios



## 🌟 주요 기능

- 사용자 회원가입 / 로그인 (JWT 인증 연동)
- 상품 목록 및 상세 조회
- 장바구니 담기, 수량 조절, 삭제
- 주문 및 결제 페이지
- 마이페이지 (주문 내역 확인, 회원 정보 수정)
- 관리자 상품 등록 / 수정 / 삭제 페이지



## 🚀 프로젝트 설정 및 실행

### 설치

```bash
npm install
```

### 실행

```bash
npm start
